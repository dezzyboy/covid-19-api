Dashboard can be found here [2019-covid-19-dashbaord](https://github.com/dezzyboy/2019-covid-19).

## Build Setup

``` bash
# set consumer keys and port
$ cp .env.example .env

# install dependencies
$ npm install

# serve at localhost:3000
$ npm start

# start cron
$ node runner
```
